Implementasi Missing Values pada Algoritma KNN#

  • Pengertian Algoritma KNN

Algoritma K-Nearest Neighbors (KNN) adalah salah satu algoritma dasar dalam machine learning yang digunakan untuk masalah klasifikasi dan regresi. Algoritma ini bekerja dengan cara mengidentifikasi kelompok-kelompok data yang serupa dalam ruang fitur. KNN juga berfungsi mengklasifikasikan objek berdasarkan mayoritas label kelas dari K tetangga terdekatnya.

  • Menghitung manual

  • Implementasikan pada missing values pada data yang dikosongkan

  • Menggunakan library scikit learn sklearn.impute.KNNImputer (https://scikit-learn.org/stable/modules/generated/sklearn.impute.KNNImputer.html)

Rumus Euclidean Distance $\( d(x,y)=\sqrt(\Sigma(xi-yi)^2)\)$

  • Langkah - langkah Algoritma KNN

  1. menentukan nilai k

  1. menghitung jarak antara data ujih dengan data latih

  1. mengurutkan dari yang terkecil

  1. menambahkan kolom yang terdekat dan dibagi dengan jumlah yang ditambahkan

Deteksi OUTLIER()#

  • Pengertian Deteksi OUTLIER()

Suatu metode yang digunakan dalam analisis data untuk mengidentifikasi titik-titik data yang dianggap sebagai outlier atau anomali. Outlier adalah nilai yang secara signifikan berbeda dari sebagian besar data lainnya dalam sebuah dataset. Outlier dapat mengindikasikan kesalahan dalam pengumpulan data, variasi alami yang ekstrim, atau kejadian yang tidak biasa.

K-distance (dist_K) adalah jarak ke tetangga terdekat ke K. K-neighbors dilambahkan dengan Nk(A) mencakup himpunan titik yang terletak di dalam atau pada lingkaran berjari-jari jarak K. K-tetangga bisa lebih dari atau sama dengan nilai K.

Rumus hitung jarak Manhattan

\[d(x,y) = \sqrt{\sum_{i=1}^{n}(x_i - y_i)}\]

  • contoh proses manual (Data berbeda dengan 3 fitur (3 dimensi) dengan jumlah data terserah)

  • implementasikan ke Python

Program Perhitungan Gaussian Naive bayes#

import numpy as np
import pandas as pd
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import accuracy_score
import csv

# Load the iris dataset
iris = load_iris()
X = iris.data
y = iris.target

# Initialize the Naive Bayes classifier
nb_classifier = GaussianNB()

# Split the data into training and test sets (80% training, 20% test)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=0)

# Train the classifier
nb_classifier.fit(X_train, y_train)

# Predict the classes for the training set
y_test_pred = nb_classifier.predict(X_test)

# Combine features and target into one list
train_data = np.column_stack((X_train, y_train))

# Write the training data to a CSV file
with open('training_data.csv', mode='w', newline='') as file:
    writer = csv.writer(file)
    # Write header
    writer.writerow(iris.feature_names + ['Target'])
    # Write data
    writer.writerows(train_data)

# Calculate the accuracy of the classifier on the training set
training_accuracy = accuracy_score(y_test, y_test_pred)
print("Training Accuracy:", training_accuracy)

# Sample data for prediction
sample = np.array([[5.3, 3.4, 4.2, 1.1]])

# Predict the class for the sample
prediction = nb_classifier.predict(sample)
print('Prediction:', prediction)
